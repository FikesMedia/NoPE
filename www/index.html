Node.js Powershell Application


Use this to create a new Powershell Web Application.

All static files are hosted out of /www folder in root.

GET Powershell scripts located in ps1/get are processed
using the "/ps1/get/:script" router rendering any output.

POST Powershell scripts located in ps1/post are processed
using the "/ps1/post/:script" router rendering any output.

Authentication Powershell scripts located in ps1/auth are processed
using the "/ps1/auth/:script" router rendering any output.

All data sent to Powershell scripts must be JSON formatted.
GET data must be JSON and sent as DATA for processing.

Standalone Node.js 14.9.0.0 is included for testing.
